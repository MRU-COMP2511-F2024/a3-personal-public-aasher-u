Stock Predictor

An advanced stock prediction model using machine learning algorithms to forecast stock trends with accuracy. After watching a video from 'Michael Reeves' where his fish predicts the stock to invest in, I wanted to try building a machine learning project myself. Although, it was not easy to learn, the coding part was surprisingly short for me. This project is still a work in progress, since i want to add more features (e.g. price trends, trading volumes, and real-time prediction) The first image shows the stock price predicted for Apple.

A responsive portfolio website showcasing skills, projects, and achievements with clean design and functionality. One challenge I faced building this website was the audio integration part. Personal Portfolio v1

A portfolio showcasing my skills, projects, and experiences. This project is yet to be finished, as I feel it is quite empty still. Building this website taught me alot about React and how it can be used without HTML. One of the more difficult part of this website was building the typewriter animation, as I used a library called "GSAP." Trying to figure out the timing with each letter made it harder, but eventually with some trial and error I got it to work with any sentence (one of my proudest moments). This website has its own domain, where I used AWS to deploy it.

TechWave

A website for a company specializing in web development and e-commerce solutions. This was a duo project, with Jan Pizarras. One of the most challenging parts of this project was making sure it was accessible. For the first version of the project we had to make a table about our services, but it was not as clean of a design we were hoping. The service page was disproportionate and too long for what we liked, so adding a 'Scroll to Top' button really helped. For the second version, we decided to go for a grid layout (sort of what I have on this page), which gave it a nice clean look.
